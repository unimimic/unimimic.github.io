實作在 Docker 環境中使用 GPU

Bunny Lee included in 技術筆記
 2023-10-15  293 words   2 minutes 
Contents
  • 安裝Docker
  • 安裝Nvidia Driver
  • 安裝NVIDIA Container Toolkit
  • 安裝NVIDIA Container Toolkit (nvidia-docker2)

以下主要有三大部分，有安裝Docker、安裝Nvidia Driver、安裝NVIDIA Container Toolkit。其中NVIDIA Container Toolkit 有兩種方法，挑其中一種進行安裝即可。

安裝Docker

如有需要更詳細的安裝，可參考官方教學。

  1. 移除舊版Docker (可選)

    1

    sudo apt-get remove docker docker-engine docker.io containerd runc

  2. 更新apt安裝包以及所需的相關套件

    1
2
3
4
5
6
7

    sudo apt-get update

sudo apt-get install \
    ca-certificates \
    curl \
    gnupg \
    lsb-release

  3. 加入docker的官方GPG Key

    1
2

    sudo mkdir -m 0755 -p /etc/apt/keyrings
curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo gpg --dearmor -o /etc/apt/keyrings/docker.gpg

  4. 設置repository

    1
2
3

    echo \
  "deb [arch=$(dpkg --print-architecture) signed-by=/etc/apt/keyrings/docker.gpg] https://download.docker.com/linux/ubuntu \
  $(lsb_release -cs) stable" | sudo tee /etc/apt/sources.list.d/docker.list > /dev/null

  5. 更新apt安裝包

    1

    sudo apt-get update

  6. 安裝docker

    1

    sudo apt-get install docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin

  7. 確認是否安裝成功

    1

    sudo docker run hello-world

安裝Nvidia Driver

  1. 移除舊的Nvidia Driver

    1
2
3
4

    sudo apt-get purge nvidia*
sudo apt-get autoremove
sudo apt-get autoclean
sudo rm -rf /usr/local/cuda*

  2. 查詢安裝版本，點此連結到官網查詢。

  3. 安裝Nvidia Driver

    1
2
3

    sudo apt update
sudo apt install nvidia-utils-<版號前三碼>
sudo apt install nvidia-driver-<版號前三碼>

  4. 重啟電腦

    1

    sudo reboot

  5. 確認安裝成功

    1

    nvidia-smi

安裝NVIDIA Container Toolkit

如有需要更詳細的安裝，可參考官方教學。

  1. 設定repo以及GPG key

    1
2
3
4
5

    distribution=$(. /etc/os-release;echo $ID$VERSION_ID) \
      && curl -fsSL https://nvidia.github.io/libnvidia-container/gpgkey | sudo gpg --dearmor -o /usr/share/keyrings/nvidia-container-toolkit-keyring.gpg \
      && curl -s -L https://nvidia.github.io/libnvidia-container/$distribution/libnvidia-container.list | \
            sed 's#deb https://#deb [signed-by=/usr/share/keyrings/nvidia-container-toolkit-keyring.gpg] https://#g' | \
            sudo tee /etc/apt/sources.list.d/nvidia-container-toolkit.list

  2. 安裝 nvidia-container-toolkit

    1
2

    sudo apt-get update
sudo apt-get install -y nvidia-container-toolkit

  3. 配置 Docker daemon

    1

    sudo nvidia-ctk runtime configure --runtime=docker

  4. 重啟 Docker

    1

    sudo systemctl restart docker

  5. 測試是否安裝成功

    1

    sudo docker run --rm --runtime=nvidia --gpus all nvidia/cuda:11.6.2-base-ubuntu20.04 nvidia-smi

安裝NVIDIA Container Toolkit (nvidia-docker2)

  1. 設定repo以及GPG key

    1
2
3

    distribution=$(. /etc/os-release;echo $ID$VERSION_ID) \
   && curl -s -L https://nvidia.github.io/nvidia-docker/gpgkey | sudo apt-key add - \
   && curl -s -L https://nvidia.github.io/nvidia-docker/$distribution/nvidia-docker.list | sudo tee /etc/apt/sources.list.d/nvidia-docker.list

  2. 安裝 nvidia-docker2

    1
2

    sudo apt-get update
sudo apt-get install -y nvidia-docker2

  3. 重啟 Docker

    1

    sudo systemctl restart docker

  4. 測試是否安裝成功

    1

    sudo docker run --rm --gpus all nvidia/cuda:11.6.2-base-ubuntu20.04 nvidia-smi
